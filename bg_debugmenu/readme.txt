-- [. About .] --------------------------------------------------------------------------

- Support for 18 unique buttons.
- Toggle and Press buttons.
- Configure state and execute functions for each button
- Fits straight into the game update loop and can run with the game active in the background


-- [. Future Plans .] -------------------------------------------------------------------

The debug menu has room for expansion:
 - Adding dropdown/select boxes
 - Cleaning up some of the magic numbers around menu button placement
 - Adding tabs/windows to incrase number of available buttons
