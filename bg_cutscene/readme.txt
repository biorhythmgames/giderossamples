-- [. About .] --------------------------------------------------------------------------

- Light weight cutscene framework
- Demonstrates play back of scripted text dialog between characters
- Can be extended to work off time based scripted events
- Update loop allows for general game components to be updated during the cutscene


-- [. Future Plans .] -------------------------------------------------------------------

There are no plans to extend this demo at the moment. If people are interested to see the
cut scene system working based on timed events, I may release an update to show exactly how 
that can be achieved.
